# Basic Todo App

A basic todo app made with Vue.js.

Features:

- Create task
- Delete task
- Edit task
- Mark/unmark task as done
- Clear whole list
- Darkmode/lightmode

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your tests

```
npm run test
```

### Lints and fixes files

```
npm run lint
```
